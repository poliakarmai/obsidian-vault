# TREK — самохостируемый планировщик путешествий с ИИ

Open-source (AGPL-3.0), Docker-развёртывание.

## Ключевое

- MCP-сервер: 150+ инструментов для AI-агентов
- Совместная работа в реальном времени (WebSocket)
- Карты (Leaflet/Mapbox), бюджеты, списки вещей
- JWT/OAuth/Passkeys аутентификация
- PWA (мобильное приложение без App Store)
- Бесплатно (только ресурсы сервера)

## Use case для Hermes

- Тенантам как фича: «спланируй поездку через ИИ»
- Вахта/командировки: логистика с AI-ассистентом
- MCP-интеграция: Hermes-агенты напрямую управляют поездками

## Вердикт

Отложено. Интересно, но не приоритетно. Вернуться когда будет спрос от тенантов или для личного использования.
