# @Apolaibot — 6 проектных кейсов

Готовые сценарии для демонстрации возможностей AI-агента.
Адаптировано из Claude Code: Полное руководство (Мукминов, 2026).

## Кейс 1: SaaS-платформа

**Задача:** спроектировать и запустить MVP SaaS за выходные.
**Скиллы:** архитектура, генерация кода, тесты, деплой.

**Как @Apolaibot помогает:**
- «Спроектируй REST API для сервиса рассылок»
- «Сгенерируй Docker-образ с нуля»
- «Напиши интеграционные тесты для платёжного модуля»
- «Настрой CI/CD в GitHub Actions»

**Демо-сценарий:** «Я делаю SaaS для учёта рабочего времени. Нужен план архитектуры и первый прототип на FastAPI.»
