# Second Brain AI Assistant Course

**GitHub:** https://github.com/decodingai-magazine/second-brain-ai-assistant-course  
**Лицензия:** MIT  
**Авторы:** Decoding AI (decodingai.substack.com)

Курс для разработчиков по созданию AI-ассистента для личной базы знаний (Second Brain) с LLM, агентами, RAG и LLMOps.

## Структура (6 модулей)

1. **Архитектура** — общая архитектура ассистента
2. **ETL** — извлечение и трансформация данных из источников
3. **Тонкая настройка (Fine-tuning)** — дообучение модели под задачу
4. **Развёртывание (Deployment)** — продакшен-пайплайн
5. **RAG** — retrieval-augmented generation
6. **LLMOps** — мониторинг, эксперименты, CI/CD для LLM

## Особенности

- Разделение на ML/данные-пайплайны (офлайн) и инференс-пайплайн (онлайн)
- Поддержка **Notion** как источника базы знаний (публичный снепшот — аккаунт не нужен)
- Готовый код, документация модулей, `uv`/`ruff`, Docker-инфраструктура
