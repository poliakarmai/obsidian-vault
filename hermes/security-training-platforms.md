## Рекомендации по порядку

1. **Начать:** PicoCTF → OverTheWire (Bandit) → TryHackMe (Junior Pentester path)
2. **Web:** PortSwigger Academy → OWASP Juice Shop → Hacker101
3. **API:** crAPI → APIsec University
4. **Машины:** HTB (Starting Point) → VulnHub → HTB (Machines)
5. **Крипта:** CryptoHack
