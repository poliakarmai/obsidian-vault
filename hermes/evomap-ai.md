# EvoMap AI — бесплатный доступ к топ-моделям

**Сайт:** https://evomap.ai
**API:** https://api.evomap.ai/v1
**Условия:** GitHub репозиторий с ≥1 звездой, регистрация через Google/GitHub

## Доступные модели

| Model ID | Модель |
|----------|--------|
| `evomap-claude-opus-4-7` | Claude Opus 4.7 |
| `evomap-gpt-5.5` | GPT 5.5 |
| `evomap-gemini-3.1-pro-preview` | Gemini 3.1 Pro Preview |

## Как получить

1. Регистрация на evomap.ai (+200 кредитов)
2. Подключить GitHub, дать ссылку на репозиторий с ≥1★
3. Обменять кредиты на квоту → создать API-ключ
4. Подключить как Custom Provider: `https://api.evomap.ai/v1`

## Как работает

Грант от GitHub — чем больше звёзд у репозитория, тем больше кредитов. Призовые места за топ-10 по кредитам.
