# Claude Code — шпаргалка (выжимка)

Источник: Ранас Мукминов, «Claude Code: Полное руководство», 2026
Файл: CC_шпаргалка.pdf

## Ключевые идеи

### Claude Code — это агент, а не автодополнение
- Работает в терминале как полноценный участник разработки
- Читает файлы, пишет код, запускает тесты, работает с git
- Анализирует ошибки, управляет зависимостями
- «Живёт» в терминале и имеет доступ к проекту

### Constitutional AI (CAI)
- Модель обучается следовать «конституции» — набору принципов на естественном языке
- Не просто генерирует текст, а делает это ответственно
- Это ровно то, что у нас: `hermes-constitution`

### Структура взаимодействия
- Часть I: установка, первые шаги, архитектура
- Часть II: практика (код, рефакторинг, отладка, тесты, БД, DevOps, Git)
- Часть III: продвинутые техники (MCP, автоматизация, безопасность, API)
- Часть IV: реальные проекты (SaaS, мобильное, CLI, Telegram-бот, Data Pipeline, микросервисы)
- Часть V: команда, этика, оптимизация, будущее

### Что можно взять
- **Constitutional AI как отдельный слой** — проверка каждого действия агента на соответствие принципам
- **Проектный подход** — в книге 6 полных проектов от начала до конца, можно адаптировать как кейсы для @Apolaibot
- **Командная работа** — раздел про внедрение AI в команду, стандартизацию, масштабирование
