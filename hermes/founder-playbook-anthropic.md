## 4 стадии AI-native стартапа

| Стадия | Что делает Claude Code | Что для @Apolaibot |
|--------|----------------------|-------------------|
| **1. Идея** | Валидация гипотез, анализ рынка, промпты для customer discovery | Каталог скиллов: маркетолог, продажи |
| **2. MVP** | Генерация кода, прототипирование, тесты | Кейс «SaaS за выходные», Sandbox |
| **3. Запуск** | Деплой, CI/CD, документация, онбординг | Демо-бот → Pro-тенант, автоматизация |
| **4. Масштаб** | Рефакторинг, мониторинг, команда | Enterprise-тариф, whitelabel |

## 9 продуктовых возможностей

1. AI-агенты для вертикальных рынков (юристы, врачи, бухгалтеры) — **это мы**
2. Кодовые ассистенты нового поколения (не автодополнение, а full-stack)
3. Инструменты для работы с данными (CSV → инсайты)
4. Документооборот с AI (контракты, письма, отчёты)
5. Образовательные AI-тьюторы (тесты, планы, проверка)
6. Креативные инструменты (дизайн, контент, маркетинг)
7. DevOps-автоматизация (мониторинг, алерты, деплой)
8. AI для найма (скрининг резюме, вопросы для интервью)
9. Локализация и перевод (мультиязычный контент)
