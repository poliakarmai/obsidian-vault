# Claude Code SKILL: YouTube Fetcher → Markdown

Инструмент для конвертации YouTube-роликов в Obsidian-ready Markdown-заметки.

## Как работает

```
YouTube URL → Claude Code SKILL → Markdown (Obsidian-ready)
```

Автоматически:
- Извлекает транскрипт/subtitles
- Формирует YAML frontmatter (title, date, tags)
- Структурирует в Markdown с заголовками, списками, код-блоками
- Сохраняет готовую заметку, оптимизированную под Obsidian

## Применимость к Hermes

У нас есть свои инструменты для этого:
- `youtube-content` скилл — транскрипты через yt-dlp + whisper
- `obsidian` скилл — создание заметок

Claude Code SKILL может быть альтернативой или дополнением. Проверить при случае.
