# Apache Burr: надёжные AI-агенты на чистом Python

**Источник:** [ai-uchi.ru](https://ai-uchi.ru/articles/apache-burr-nadezhnyye-ai-agenty/)
**Дата:** 12 июня 2026
**Лицензия:** Apache 2.0 (инкубация)

---

## Главная идея

Apache Burr — легковесный Python-фреймворк для построения предсказуемых, наблюдаемых и тестируемых AI-агентов. Модель конечных автоматов: Action → State → Transition. Иммутабельное состояние, встроенный UI для трассировки, чистые функции.

## Ключевые концепции

- **Action** — атомарный шаг, Python-функция с `@action`
- **State** — иммутабельный объект между действиями (каждый шаг возвращает новое состояние)
- **Transition** — правила переходов на чистом Python, без DSL/YAML
- **Tracker** — сохранение трассировок (локально, БД, кастомные бэкенды)

## Преимущества

1. **Управление состоянием** — иммутабельное, автосохранение. При сбое на шаге 7 — перезапуск с шага 7, не с начала
2. **Встроенный UI** — трассировка в реальном времени + OpenTelemetry
3. **Тестируемость** — каждое действие чистая функция `State → State`, юнит-тесты без мок-серверов

## Сравнение с аналогами

| Критерий | Apache Burr | LangChain/LangGraph | CrewAI/AutoGen |
|----------|-------------|---------------------|----------------|
| Парадигма | State machine | Chain/Graph | Role-based multi-agent |
| Кривая обучения | Низкая (чистый Python) | Высокая | Низкая для старта |
| Наблюдаемость | Встроенный UI + OTEL | LangSmith (платный) | Сторонние |
| Тестируемость | Высокая | Средняя | Низкая |
| Зрелость | Инкубация (2024) | Зрелый (2022) | Активно развивается |

## Применимость к Hermes

| Концепция | Burr | Hermes |
|-----------|------|--------|
| Модель | Конечный автомат (граф действий) | Цикл агента + инструменты |
| Состояние | Иммутабельное, сохраняемое | Сессионное (SQLite) |
| Трассировка | Встроенный UI | session_search + логи |
| Тестирование | Чистые функции | Через cron/мониторинг |
