# Искра (iskrabot.ru) — конкурентный анализ

> Российский AI-ассистент, аналог Perplexity/ChatGPT. Агенты, веб-поиск, проекты, on-premise.

## Продукт
- **Сайт:** https://iskrabot.ru
- **Облако:** https://cloud.iskrabot.ru
- **Модель:** Qwen 3.5 Plus (флагшип)
- **Тарифы:** Старт (бесплатно), Базовый (690₽), Про (1990₽), Эксперт (5990₽)
- **Приложения:** iOS, macOS, Linux (.deb), Windows (.exe), мобильная PWA
- **Агентская система:** Antomic (собственная разработка)

## Ключевые фичи по релизам

### Агенты
- Собственная агентская система Antomic (быстрее и дешевле аналогов)
- Агент делегирует мини-агентам сложные задачи
- Ожидание освободившегося агента (не отказ)
- Агент задаёт структурированные вопросы с вариантами ответов
- Python и Node.js в инструментарии агента
- Агент видит общие файлы проекта
- Живой вывод агента (streaming tool output)
- Агент спрашивает разрешение на действия (approvals)
- Карта документа для больших файлов (частичное чтение)

### Пространства / Проекты
- Пространства — общие чаты, участники, файлы
- @Искра в общем чате — читает все документы и отвечает
- Проекты и Базы знаний
- Папки с правами доступа (пользователи, группы, организация)

### Файлы
- PDF, DOCX, XLSX, PPTX, EPUB, RTF, CSV — чтение
- Картинки внутри документов — извлечение + описание
- Старые .doc — тоже читает
- HEIC, SVG, AVIF, TIFF, BMP — авто-конвертация
- Превью файлов прямо в чате
- Скачивание в PDF/Word/Excel из текстовых форматов

### База знаний
- RAG по документам организации
- Цитаты и ссылки на источники
- Поиск разнообразит источники (не один документ)
- Авто-именование файлов (ИИ придумывает название)
- Уведомления о доступе к папке

### Поиск
- Несколько параллельных поисковых запросов
- Результаты видны в процессе

### Расписание (cron)
- Задачи по расписанию: ежедневно, по будням, раз в неделю
- Каждый запуск — отдельный чат
- Уведомления на почту и в приложение
- Админка с дашбордом и историей запусков

### Интерфейс
- Стриминг с автопереподключением (не теряется при обрыве)
- Форк чата от любого сообщения
- Инкогнито-режим (временный чат)
- Скачать чат архивом
- Оценки 👍/👎 на каждом сообщении
- Панель артефактов (файлы и ценные ответы)
- Тёмная/светлая/системная тема
- Поиск по всем чатам

### Организации / B2B
- Переключение между организациями
- @Искра в общем чате команды
- Приглашения по ссылке
- Админка: просмотр чатов, управление пространствами, мониторинг
- 40+ параметров через админку без перезагрузки

### Онбординг
- Выбор роли, задач и целей облаком тегов
- Искра пишет первой (приветствие + подсказка)
- Инструктор (ИИ-инструктор)

### Безопасность
- Captcha на входе
- 2FA для админов (Google Authenticator / Яндекс Ключ)
- Защита от ботов
- ИНН больше не блокируется как паспортные данные

### Биллинг
- 4 тарифа (Старт/Базовый/Про/Эксперт)
- Промокоды
- Организации могут оплачивать сотрудникам
- Предупреждение при 80% лимита токенов, блокировка при 100%

### On-premise
- Свой лендинг (ZIP-загрузка)
- Массовое создание пользователей
- Свои шаблоны документов
- Брендинг под компанию
- Аудит-лог изменений

### Агентская файловая система
- Версионирование файлов
- Optimistic concurrency
- Lock на путь
- Привязка артефактов к сообщениям
- Rollout/rollback runbook

### Прочее
- Вход через Яндекс ID и VK ID
- iOS/Android приложения
- Десктопные приложения
- Презентации (PPTX)
- Англоязычный интерфейс
- Мониторинг здоровья платформы
- Устойчивость к сетевым сбоям (ретрай)
