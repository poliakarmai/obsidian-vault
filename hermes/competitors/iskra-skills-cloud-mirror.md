# Искра: Навыки + Облачное зеркало

**Суть:** пользователи и организации создают навыки (промпты или программы). 
Искра сама выбирает подходящий навык. MCPHub — изолированная песочница для запуска.

**Наш статус:** 70% готово
- ✅ Skills (108, tiered)
- ✅ skill-sandbox (Podman)
- ✅ skill-sync (авто-синхронизация)
- ✅ MCP-серверы
- ❌ Тенанты не могут создавать навыки
- ❌ Нет UI/интерфейса для загрузки

**Что нужно:** разрешить tenant'ам создавать навыки в `skills.local/` 
→ review pipeline → импорт в base (skill-sync v3 --pull уже умеет)

Агенты работают с файлами напрямую. Двусторонняя синхронизация.

**Наш статус:** 0%
- ❌ Нет десктоп-клиента

**Когда делать:** не в ближайшем скоупе. Требует отдельного продукта.
