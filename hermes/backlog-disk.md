# Бэклог: завтра

## Быстрое заполнение диска
- **Проблема:** диск с 73% до 96% за день при установке whisper
- **Причины:** torch 1.2G, nvidia/cuda 2.7G, pip cache 4G, state.db WAL 520M
- **Решения:**
  1. Мониторинг диска с алертом при >80% (уже есть cron disk-monitor)
  2. Авто-очистка pip cache после установок
  3. state.db VACUUM по расписанию
  4. Ограничить размер логов (cron/output уже >7д чистится)
  5. Whisper → использовать API вместо локальной модели
