# Учебный бот для специалистов НК

## Проблема

Специалисты неразрушающего контроля готовятся к аттестации по разрозненным материалам: бумажные билеты, PDF-файлы ГОСТов, ссылки на форумы. Нет одного места где можно: а) пройти билеты по своему методу, б) задать уточняющий вопрос и получить ответ с цитатой из ГОСТа.

## Рекомендуемое направление

**Telegram-бот «НК-Репетитор»** — вариант B (средний).

### Что внутри

```
Пользователь → выбирает метод НК (РК/УК/ВК/ВИК/МК/ПВК/ТК)
    ├── Режим «Билеты»: случайный билет → вопрос → ответ → следующий
    ├── Режим «Экзамен»: N вопросов, таймер, результат
    └── Режим «AI-вопрос»: свободный вопрос → RAG-поиск по ГОСТам → ответ с цитатой
```

### Технический стек

| Компонент | Решение |
|-----------|---------|
| Фронтенд | Telegram-бот (aiogram) |
| База билетов | SQLite / JSON |
| Документы | PDF ГОСТов, учебников → OCR → Markdown в Obsidian vault |
| AI-агент | RAG: BM25 поиск по vault → DeepSeek с цитатами |
| Платёж | Telegram Stars |
| Хостинг | VDS (текущая) |

### Методы НК (приоритет)

| Метод | Расшифровка | Массовость |
|-------|------------|-----------|
| РК | Радиографический контроль | 🔴 высокая |
| УК | Ультразвуковой контроль | 🔴 высокая |
| ВИК | Визуально-измерительный | 🟡 средняя |
| ВК | Визуальный контроль | 🟡 средняя |
| МК | Магнитный контроль | 🟢 низкая |
| ПВК | Контроль проникающими веществами | 🟢 низкая |
| ТК | Тепловой контроль | 🟢 низкая |

## Ключевые допущения

- Билеты можно собрать из открытых источников — риск: 🟡 (проверить авторские права)
- ГОСТы доступны для образовательного использования — риск: 🟢
- Спрос есть: >5000 специалистов НК аттестуются ежегодно — риск: 🟡 (проверить)
- AI + RAG даст качество ответов >80% — риск: 🔴 (нужен строгий промпт)

## MVP

1. **1 метод НК = РК** (самый массовый)
2. **50 билетов** (реальные, из открытых источников)
3. **5 ГОСТов** в базе (РК-специфичные)
4. **Режим «Билеты» + «AI-вопрос»**
5. **Оплата Stars** — ~100-200 звёзд/месяц (≈300-600₽)
6. **Запуск на 10 тестовых пользователях**

## НЕ делаем

- Веб-интерфейс (только Telegram)
- Личный кабинет со статистикой (v2)
- Международные стандарты (только РФ)
- Интеграцию с НКЦ (сложно, v3)
- Все 7 методов сразу (начинаем с 1)
