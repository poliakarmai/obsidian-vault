# Grafana HTTPS — не грузится в браузере

## Что сделано

- Nginx: HTTP (80) → авто-редирект на HTTPS (8443)
- SSL: самоподписанный сертификат (SAN: IP:2.27.48.142)
- Grafana v13.0.2 работает, порт 3000
- API отвечает: `/api/health → ok`, логин через JSON API работает
- Curl-тесты проходят: логин → Set-Cookie → дашборд отдаётся (200)

## Симптом

Браузер (Chrome/Firefox/мобильные) — страница логина грузится, но после ввода admin/admin не заходит. "Грузит и не заходит."

## Гипотезы

1. **Самоподписанный сертификат** — браузеры блокируют AJAX/fetch-запросы к самоподписанным сертификатам даже после ручного исключения
2. **Нестандартный порт 8443** — мобильные браузеры/корпоративные сети могут блокировать
3. **CORS/ CSP** — Grafana v13 может требовать strict-origin

## Что нужно попробовать

- [ ] Let's Encrypt сертификат (нужен домен, не IP)
- [ ] Порт 443 вместо 8443
- [ ] Проверить с разных устройств/сетей
- [ ] Проверить консоль браузера на ошибки
