## Что делать если потерял телефон с 2FA

1. При входе на GitHub нажать «Use recovery code»
2. Ввести ЛЮБОЙ неиспользованный код из списка выше
3. После входа — сразу настроить новый TOTP
4. Сгенерировать новые коды, сохранить сюда заново
