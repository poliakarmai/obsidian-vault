# Hermes Automation Blueprints

Новая фича Hermes Agent — шаблоны для cron-автоматизаций. Превращает повторяющийся процесс в форму, кнопку или короткую команду.

## Как было раньше

```
cronjob create: расписание + длинный промпт + доставка + навыки + скрипт + условия молчания
```

Пример: «каждый будний день в 18:00 собери новостной дайджест, проверь источники, убери повторы»

## Как стало

Упаковка в **Blueprint** — готовый сценарий с понятными полями:

- В GUI: форма → выбрать шаблон → тема/время/канал → Schedule it
- В Telegram/CLI/TUI: `/blueprint news-digest`
- Hermes сам спрашивает недостающие параметры, показывает что создаст, просит подтверждение

## Готовые пресеты

| Blueprint | Назначение |
|-----------|-----------|
| 🔵 Утренний брифинг | Ежедневная сводка |
| 🔵 Мониторинг важной почты | Фильтр по ключевым отправителям |
| 🔵 Еженедельный обзор | Weekly digest |
| 🔵 Дайджест по теме | Тематический мониторинг |
| 🔵 Напоминания и привычки | Персональные напоминания |
| 🔵 Проверки деплоя/алертов/PR/CI | Webhook-сценарии |

## Кастомные Blueprints

Это обычный Hermes Skill с блоком `metadata.hermes.blueprint` в SKILL.md.

**Итог:** повторяемый процесс → кнопка/форма/короткая команда.

## Применимость к нам

- Утренняя сводка (погода Бованенково + дедлайны) — кандидат на `morning-briefing` blueprint
- Еженедельный обзор по трейдингу — `weekly-review`
- Мониторинг VPN-сервера — `health-check`
- Дайджест по нефтегазу/коррозии — `topic-digest`

**Блокер:** нужна версия Hermes новее v0.17.0 (мы отстаём на 329 коммитов).
